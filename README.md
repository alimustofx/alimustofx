## 🧭 The Short Version

I'm a school IT operator and self-taught infrastructure engineer based in **East Java, Indonesia**.
My day job is keeping education technology running — student data systems, school information portals, network security, and the servers underneath all of it.

At night, I write Laravel applications and experiment with virtualization.

> *"Technology should simplify education, not complicate it."*

## 📐 How I Think About Systems

```
School Network
│
├── Proxmox Hypervisor
│   ├── VM: Ubuntu Server (Laravel Apps)
│   │   ├── Student Information System
│   │   ├── Data Verification Portal
│   │   └── Internal Tools
│   ├── VM: Database Server (MySQL)
│   └── CT: Monitoring Stack
│
├── Edge: Cloudflare
│   ├── DNS · DDoS Protection · Zero Trust
│   └── Tunnel → Internal Services
│
└── End Users: Teachers · Staff · Students
```

I don't just deploy apps — I design the environment they live in.
