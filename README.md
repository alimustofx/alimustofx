## 🧭 &nbsp;The Short Version

I'm a school IT operator and self-taught infrastructure engineer based in **East Java, Indonesia**.
My day job is keeping education technology running — student data systems, school portals, network security, and the servers underneath all of it.

At night I write **Laravel** applications and push my **Proxmox** clusters further.

> *"Technology should simplify education, not complicate it."*
